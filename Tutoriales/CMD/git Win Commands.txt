Git - Tecnologia (sistema de control de versiones).

Github - Hosting de repositorios Git.



	Commands

* git --version .. get git version

* ls -la ... show files change



	Create folder and go there!

* mkdir fileName ... create new folder

* cd rootName... move on directories

* clear ... clear screen

	
	Create and management repository git

* git init ... create repository git

* git add filename.ext ... add file to repository git

* git status ... show data status of repository git

* git log ... show info all commit
	git log --oneline ... show info in one line

* git commit -m "Primer commit con Git" ... Do commit

* git commit { open V console editor } ...
	Esc + i ... edit file
	Esc ... to exit
	:wq ... write quit - save commit with V console editor and exit to V console
	:q ... quit - exit to V console



	Upload local repository to remote repository on Github


* git remote add alias https://github.com/Backig/ReciclApp.git

* git remote -v ... view list all remotes


	Push data to remote repository git

* git push -u alias master

* git clone https://github.com/Backig/ReciclApp.git


* git remote rename origin newName ... change repository git name

	Configure your settings on repository git


* git config --global user.name "Your name here" ... set name for git to use when you commit

* git config --global user.email "someone@example.com" ... set email for git to use when you commit
